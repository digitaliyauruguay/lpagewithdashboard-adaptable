========================================

🆕 CREAR NUEVO PRESET:

1. Copiar archivo existente:
   cp src/config/presets/veterinaria.js src/config/presets/mi-rubro.js

2. Editar todos los valores en mi-rubro.js

3. Importar en activePreset.js:
   import { miRubroPreset } from './presets/mi-rubro.js';

4. Activar:
   export const config = miRubroPreset;

========================================

🆘 PROBLEMAS COMUNES:

❌ "No guarda leads"
→ Verificar USE_MOCK = false en /src/lib/airtable.js
→ Verificar credenciales en preset

❌ "No puedo logearme"
→ Ver credenciales en config.admin del preset activo
→ Limpiar localStorage del navegador

❌ "Carousel no funciona"
→ Verificar que slick-carousel esté instalado
→ Refrescar navegador
